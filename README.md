# 智能客服项目

## 项目概述

这是一个基于 LangGraph 和向量检索的智能客服系统，支持自然语言对话、知识库检索和工单管理。

## 项目结构

- `knowledge_base/`：存放示例知识文档，用于构建向量检索数据库。
- `chroma_db/`：向量库持久化目录，运行时自动生成。
- `src/`：项目核心代码目录。
  - `app.py`：Streamlit 前端应用入口。
  - `agent.py`：LangGraph Agent 核心逻辑。
  - `build_vectorstore.py`：向量库构建脚本。
  - `tools.py`：工具函数和类。

## 快速开始

### 环境准备

1. 创建并激活 Python 虚拟环境：
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Linux/Mac
   source venv/bin/activate
   ```

2. 安装项目依赖：
   ```bash
   pip install -r requirements.txt
   ```

### 运行应用

启动 Streamlit 前端应用：
```bash
streamlit run src/app.py
```

### 核心组件说明

- **前端界面**：基于 Streamlit 的用户交互界面。
- **后端服务**：可选的 FastAPI 服务，用于扩展 API 接口。
- **Agent 核心**：使用 LangGraph 管理对话流程，包括意图识别、工具执行和答案生成。
- **工具模块**：
  - RAG 检索工具：基于 Chroma 向量库的知识检索。
  - 订单查询工具：模拟数据库查询功能。
  - 创建工单工具：模拟工单创建和写入。
- **嵌入模型**：使用 BAAI/bge-small 模型进行文本向量化。
